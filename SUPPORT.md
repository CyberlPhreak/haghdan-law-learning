# HaghDān Support

HaghDān is an independent SQE and England-and-Wales law learning application.

## Get help

- [Ask for technical or account help](https://github.com/CyberlPhreak/haghdan-law-learning/issues/new?template=support-request.yml)
- [Report an inaccurate, outdated or rights-sensitive item](https://github.com/CyberlPhreak/haghdan-law-learning/issues/new?template=content-error.yml)
- [Submit a privacy or deletion request](https://github.com/CyberlPhreak/haghdan-law-learning/issues/new?template=privacy-request.yml)

Support requests are handled through this repository's issue tracker. Because the repository and issues are public, never include a PIN, legal case details, privileged material, identification numbers, financial information, health information or another person's personal data. If a report requires sensitive evidence, first submit only a non-sensitive summary and request a private contact route.

## Common fixes

1. Update to the latest app version.
2. Fully close and reopen the app.
3. Confirm the device has sufficient storage.
4. If online AI is unavailable, continue with the automatic offline curriculum assistant.
5. If local progress is corrupted and cannot be recovered, use **Profile → Support → Delete account and data**. This permanently clears app-local progress.
