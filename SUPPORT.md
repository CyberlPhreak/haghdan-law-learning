# HaghDān Support

HaghDān is an independent SQE and England-and-Wales law learning application.

## Get help

- [Ask for technical or account help](https://github.com/CyberlPhreak/haghdan-law-learning/issues/new?template=support-request.yml)
- [Report an inaccurate, outdated or rights-sensitive item](https://github.com/CyberlPhreak/haghdan-law-learning/issues/new?template=content-error.yml)
- [Submit a privacy or deletion request](https://github.com/CyberlPhreak/haghdan-law-learning/issues/new?template=privacy-request.yml)

Support requests are handled through this repository's issue tracker. Because the repository and issues are public, never include a password, PIN, verification/reset link, legal case details, privileged material, identification numbers, financial information, health information or another person's personal data. If a report requires sensitive evidence, first submit only a non-sensitive summary and request a private contact route.

## Common fixes

1. Update to the latest app version.
2. Fully close and reopen the app.
3. Confirm the device has sufficient storage.
4. If verification mail is missing, check spam and use **Resend verification email**. Password recovery requires access to the registered inbox.
5. If cloud sync is unavailable, keep the app installed and retry after reconnecting; the device cache is retained.
6. If online AI is unavailable, continue with the automatic offline curriculum assistant.
7. Use **Profile → Support → Delete account and data** to permanently delete the active cloud account and associated progress, or the offline fallback account.
